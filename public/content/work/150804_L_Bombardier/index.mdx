<!--commentary-->{{< projCommentary title="Project" subtitle="Synopsis" >}}
<a class="link link--loud" href="https://momentfactory.com/home">Moment Factory</a> contacted me to design a principal element in a permanent installation for Bombardier. The brief was to come up with a visual representation for a timeline.
{{< /projCommentary >}}

<!--image-->{{< projImage filename="002*" size="large" >}}

<!--commentary-->{{< projCommentary title="Project" subtitle="Styleframes" >}}
The timeline unfolds in the form of an abstract structure inspired by architecture from Mies. The structure levitates above a grid reminiscent of a runway. Particles stream underneath representing the perpetual flow of time.
{{< /projCommentary >}}

<p>A while back I saw a series of generative artwork created using the gradient descent algorithm. Today at work I was given those images as references for a project, and I thought I could recreate the effect using particle flow with a data operator.</p>

<p>I made a rough version of the data op at work that does the job, but I thought it is a pretty neat effect so I decided to refine it a bit further and save it as a preset.</p>

<p>You can <a class="link link--loud-simple" href="https://www.dropbox.com/s/m7etsrgf7o8zmv1/Gradient%20Tracer%20%28hhx%29.zip?dl=0">download</a> the preset. Remember to use 16-bit or higher source images for better result!</p>

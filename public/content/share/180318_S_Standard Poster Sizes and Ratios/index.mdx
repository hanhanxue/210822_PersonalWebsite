<p>I have researched this topic a few times over the years. The information I wanted always seemed to be scattered around on different resources. So I've decided to consolidate all the specifications and compile this crib sheet for future reference.</p>

<p>I'm interested in all the different poster size standards, and the ratios and dimensions (both absolute and relative) associated with each.</p>

<p>The five sizes of interest from smallest to largest are: Tabloid (aka Ledger or ANSI B), Architectural C, Architectural D, One Sheet and Bus Stop.</p>
{{< /postText >}}

<h2>Tabloid (aka Ledger or ANSI B)</h2>

<ul>
    <li>Ratio (w:h): 0.6471</li>
    <li>Absolute Dimensions: 11 x 17 inches / 279 x 432 mm</li>
    <li>At 72 PPI: 792 x 1224 pixels</li>
    <li>At 150 PPI: 1650 x 2550 pixels</li>
    <li>At 300 PPI: 3300 x 5100 pixels</li>
</ul>

<h2>Architectural C</h2>

<ul>
    <li>Ratio (w:h): 0.75</li>
    <li>Absolute Dimensions: 18 x 24 inches / 457 x 610 mm</li>
    <li>At 72 PPI: 1296 x 1728 pixels</li>
    <li>At 150 PPI: 2700 x 3600 pixels</li>
    <li>At 300 PPI: 5400 x 7200 pixels</li>
</ul>

<h2>Architectural D</h2>

<ul>
    <li>Ratio (w:h): 0.6667</li>
    <li>Absolute Dimensions: 24 x 36 inches / 610 x 914 mm</li>
    <li>At 72 PPI: 1728 x 2592 pixels</li>
    <li>At 150 PPI: 3600 x 5400 pixels</li>
    <li>At 300 PPI: 7200 x 10800 pixels</li>
</ul>

<h2>One Sheet</h2>

<ul>
    <li>Ratio (w:h): 0.675</li>
    <li>Absolute Dimensions: 27 x 40 inches / 686 x 1016 mm</li>
    <li>At 72 PPI: 1944 x 2880 pixels</li>
    <li>At 150 PPI: 4050 x 6000 pixels</li>
    <li>At 300 PPI: 8100 x 12000 pixels</li>
</ul>

<h2>Bus Stop</h2>

<ul>
    <li>Ratio (w:h): 0.6667</li>
    <li>Absolute Dimensions: 40 x 60 inches / 1016 x 1524 mm</li>
    <li>At 72 PPI: 2880 x 4320 pixels</li>
    <li>At 150 PPI: 6000 x 9000 pixels</li>
    <li>At 300 PPI: 12000 x 18000 pixels</li>
</ul>

## Type Name and Specifications

### Tabloid (aka Ledger or ANSI B)

- Ratio (w:h): **0.6471**
- Absolute Dimensions: **11 x 17 inches / 279 x 432 mm**
- Resolution @ 72 PPI: **792 x 1224 pixels**
- Resolution @ 150 PPI: **1650 x 2550 pixels**
- Resolution @ 300 PPI: **3300 x 5100 pixels**

### Architectural C
- Ratio (w:h): **0.75**
- Absolute Dimensions: **18 x 24 inches / 457 x 610 mm**
- Resolution @ 72 PPI: **1296 x 1728 pixels**
- Resolution @ 150 PPI: **2700 x 3600 pixels**
- Resolution @ 300 PPI: **5400 x 7200 pixels**

### Architectural D
- Ratio (w:h): **0.6667**
- Absolute Dimensions: **24 x 36 inches / 610 x 914 mm**
- Resolution @ 72 PPI: **1728 x 2592 pixels**
- Resolution @ 150 PPI: **3600 x 5400 pixels**
- Resolution @ 300 PPI: **7200 x 10800 pixels**

### One Sheet
- Ratio (w:h): **0.675**
- Absolute Dimensions: **27 x 40 inches / 686 x 1016 mm**
- Resolution @ 72 PPI: **1944 x 2880 pixels**
- Resolution @ 150 PPI: **4050 x 6000 pixels**
- Resolution @ 300 PPI: **8100 x 12000 pixels**

### Bus Stop
- Ratio (w:h): **0.6667**
- Absolute Dimensions: **40 x 60 inches / 1016 x 1524 mm**
- Resolution @ 72 PPI: **2880 x 4320 pixels**
- Resolution @ 150 PPI: **6000 x 9000 pixels**
- Resolution @ 300 PPI: **12000 x 18000 pixels**
